# Contributing

Thanks for contributing.

## Development setup

1. Open the repository in your IDE.
2. Create a virtual environment.
3. Install dependencies from `media_scheduler/requirements.txt`.
4. Run from `media_scheduler/main.py`.

## Guidelines

- Keep changes focused and small.
- Do not mix refactors with feature changes unless necessary.
- Preserve existing function signatures unless discussed first.
- Add or update documentation when behavior changes.

## Pull requests

- Use a clear title and short description of why the change is needed.
- Include screenshots for GUI changes.
- List manual test steps.
